iScroll Widget

Requried

  • jQuery UI Core
  • Mousewheel Plugin
  • Fixed size for element (see source code of this is page)

Optional

  • Disable Text Select Plugin

Options

  • height - set height of element

Example

		    <link rel="stylesheet" href="css/iphone.css" type='text/css' media='all' />
		    <script type="text/javascript" src="js/jquery.js"></script>
		    <script type="text/javascript" src="js/jquery-ui.js"></script>
		    <script type="text/javascript" src="js/jquery.disable.text.select.js"></script>
		    <script type="text/javascript" src="js/jquery.mousewheel.js"></script>
		    <script type="text/javascript" src="js/ui/ui.iScroll.js"></script>
			<script type="text/javascript">
		    $(document).ready(function(){
				// jquery.iphone.js have autoloader, but you can use
				// alternative method of init widget
				// set height here
		    	$(".scroll").iScroll({height:200});
		    });
		    </script>
			<style type="text/css">
				.scroll {
				    /* or set height like this */
					height:200px;		
				}
			</style>
            <div class="scroll">
            Lorem ipsum ...
            </div>
